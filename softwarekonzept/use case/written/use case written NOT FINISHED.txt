Case: Upload picture
Description: A Visitor / User / Gardener / Administrator wants to upload a picture of a plant
Prerequisite: Any Planth Health ™  website's page is open
Steps:
1. The  Visitor / User / Gardener / Administrator click on the upload a picture button
2. The  Visitor / User / Gardener / Administrator gets reidrected on the camera view, where the QR code of the plant can be scanned
3. Once the QR code is scanned, the  Visitor / User / Gardener / Administrator gets redirected to the plant information page
4. Here the  Visitor / User / Gardener / Administrator click the upload new picture button
5. The  Visitor / User / Gardener / Administrator gets redirected to the device gallery where the picture to upload can be chosen
Postcondition: The picture gets added to a review list where the plant Gardener will have to approve or deny, before it gets added to the viewable gallery of the plant
Alternatives:

Classes Involved: Visitor / User / Gardener / Administrator
