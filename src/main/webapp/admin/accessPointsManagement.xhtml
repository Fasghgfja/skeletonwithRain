Plant Health - Access Points Management
About Deleting a Access Point:
You can delete a Access Point by pressing the dedicated button. When deleting a Access Point all the sensor stations connected to id will be unpaired.

About Creating a Access Point:
By Pressing the appropriate button a Dialog to create a Access Point will open.

From this panel you will be able to select Access Point ID (imput a Number) , and Location

Note: a newly created AccessPoint will be unvalidated by default.
