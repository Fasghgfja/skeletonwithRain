About Deleting a Sensor Station:
You can delete a Sensor station by pressing the dedicated button. when deleting a SS all related entities gonna be detached automatically:

Gardeners will be unassigned , plants detached , sensors deleted and measurements detached;
About Sensor Station Trashold:
the trashold determines when a alarm status will be red or green.
About Creating a sensor station:
By Pressing the appropriate button a Dialog to create a Sensor Station will open.

From this panel you will be able to select sensor-station name , description and Location

Note: a newly created SS will always have The Alarm switch set to off by default.

About Editing A Sensor Station
By Pressing the appropriate button a Dialog to edit the selected Sensor Station will open.

From this panel you will see the selected sensor-station name and will be able to: Edit the Location.

Assign and unassign the SS to Gardeners(many gardeners can work on the same sensor station if necessary).

Note: Alarm switch will be on if one of the alarm count of a sensosr station triggered it , you can manually set it to fixed from here.

About Pairing:
From the pairing Dialog it is possible to pair the sensor station to a new access point or remove the current one.

Upon initiating the pairing process the system will start to try to pair the selected sensorStation with the access point for 5 minutes.

until interrupted by pressing the dedicated button.
About The Status:
The status Icon show the alarm status and not the last measurement status. A alarm status will only be red if the alarm count is hight enough.
About The Dropdown Menu
The Dropdown menu will show the last 6 measurements(1 per type) and the Alarm status for the given type.

NOTE: even a water measurement of 100% can be Ok (green) if its not repeated (for example after a plant is irrigated)

thats the reasonw hy the status shows the alarm status and not if the displaye dmeasurement has exceeded the set tresholds on his own.
About The Details View
By clicking on the associated button you can open the DetailPage , in the detailpage you will have the possibility to:



See all measurements for the chosen Sensor Station with the corresponding Graphs.

Manage the pictures users take of the plants , Pictures will need your approval before going to the public Plant Gallery.

Edit Things like the location of the sensorStation and the plant displayed.

Remove The Current Pairing And/Or Pair to a new Access Point.

Reset the Alarm Switch .

See all Sensors of the chosen Sensor Station with their relative information

Edit Sensors bordervalues aswell as resetting the respective Alarm Counts.

More Detailed Usage infos can be found on the page itself.
