Dashboard

This is the main Page of our application, the dashboard will apeear different depending on the role of the user logged in.

  • Admin dashboard
The admin will see all 3 dashboards if he doesent remove the other roles from himself. From the Admin dashboard the Admin is able to see some top icons that display the number of registered users,access points, plants under care along with the total numbers of measurements. From his dashbord a admin can see a list of all sensor stations in the system with their sensor status and can click of them to show a bar graph showing the last 6 measuremnts(1 per type) At the end of the page the Admin is able to see a preview of the audit log that can be opened by clicking on the panle header. By clicking on the button a adminc an open the sensor station detail view.
  • Gardener Dashbord
The Gardener can also see the topboxes. On his dashbord the gardener will have a view very similar to that of a admin but only the sensor stations assigned to him will show The gardener will also see the User dashbord below
  • User Dashbord
A user on his dashbord will se the plant he is actually following and by clicking on the apposite button he will be able to see thir plant page by expanding the row the user will be able to see the last measurments for a given plant along with the sensor status.
